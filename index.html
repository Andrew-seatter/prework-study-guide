Prework Study Guide

HTML

  • always start html doc's with !DOCTYPE html
  • html elements contain a starting tag, content and an ending tag
  • the "head" element is displayed on the tab name of the webpage
  • all the contents displayed on a webpage are contained within two body tags

CSS

  • Margin property: indicates how much space you want around the outside of an element
  • Padding property: adds space inside an element
  • * pertains to every every element of the html doc

Git

  • Enter your Git notes here
  • git status: checks what branch you are currently working in
  • git checkout -b branch name: creates a new branch and moves you to working in that branch
  • git pull origin main: updates your machine to the current state of the branch ie: downloading necessary files
  • git add -A adds all changes to the current branch your working in
  • git commit -m "your description", pushes those changes to the branch

JavaScript

  • For loops are structured the same in JS as they are in C
  • JavaScript uses length functions in arrays similar to strings in C
  • console.log is the printf equivalent
  • functions are declared and recalled to repeat a block of code and structured inside {curly bracers}
